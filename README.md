## Опис
Перше домашнє завдання з курсу JS. Написано три функції для роботи з простими обчисленнями та форматуванням рядків.

## Технології / стек
- JavaScript (ES6+)
- шаблонні рядки (template literals)
- базові арифметичні операції
- робота з типами (Number.parseFloat)

## Функціонал

1. **task-1.js** — `makeTransaction(quantity, pricePerDroid)`  
   Розрахунок загальної вартості замовлення дроїдів та повернення форматованого повідомлення.

2. **task-2.js** — `getShippingMessage(country, price, deliveryFee)`  
   Підрахунок повної вартості доставки (товар + доставка) та повідомлення з назвою країни.

3. **task-3.js** — `getElementWidth(content, padding, border)`  
   Обчислення повної ширини елемента за моделлю border-box (content + padding×2 + border×2).
